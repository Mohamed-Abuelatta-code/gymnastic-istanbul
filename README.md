# Gymnastic Istanbul

موقع عرض لمشروع "جمباز إسطنبول" — صفحة ثابتة HTML تحتوي على معلومات المشروع، صور، وقسم الأهداف والقيم.

كيفية النشر عبر GitHub Pages:

- تمّ دفع الملفات إلى هذا المستودع.
- لإنشاء صفحة GitHub Pages تلقائيًا: قمت بإنشاء فرع `gh-pages` ودفعته إلى المستودع. بعد لحظات قد تصبح الصفحة متاحة على:

  `https://Mohamed-Abuelatta-code.github.io/gymnastic-istanbul/`

- إذا لم تظهر الصفحة بعد، افتح إعدادات المستودع في GitHub → Pages وتأكد أن مصدر النشر مضبوط على فرع `gh-pages` أو على `main`/`/docs` حسب تفضيلك.

ملحوظة: إذا تريدني أن أضبط اسم النطاق المخصّص أو أضيف ملف `CNAME` أخبرني.
